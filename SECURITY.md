# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do NOT open a public issue.**

Instead, please email [9169332+GeiserX@users.noreply.github.com](mailto:9169332+GeiserX@users.noreply.github.com) with details of the vulnerability.

I will acknowledge your report within 48 hours and work with you to resolve the issue before any public disclosure.
